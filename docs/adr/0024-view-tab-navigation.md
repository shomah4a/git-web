# ADR 0024: ビュー間タブナビゲーション

## ステータス

承認済み

## コンテキスト

ADR 0022 で `/tree`、ADR 0023 で `/`（worktree）を分離し、`/diff` と合わせて3つのビューが存在する。
しかし、ビュー間を遷移するための UI 導線がなく、URL を直接入力する以外に移動手段がない。

## 決定事項

### タブナビゲーションの配置

- App.vue の `<router-view>` 直上に全ページ共通のタブナビゲーションを配置する
- 現在のルートに応じてアクティブ状態をスタイリングする

### タブ構成

| タブラベル | リンク先         | 説明                                               |
| ---------- | ---------------- | -------------------------------------------------- |
| Worktree   | `/`              | worktree の現在状態（ADR 0023）                    |
| Diff       | `/diff`          | HEAD vs worktree の差分表示（ADR 0019 デフォルト） |
| Revision   | `/tree?rev=HEAD` | HEAD のツリー表示（ADR 0022）                      |

### 実装方針

- `router-link` を使用し、Vue Router のアクティブクラスでスタイリングする
- `/` の router-link は exact マッチを使い、他ページ遷移時にアクティブにならないようにする
- 既存の CSS 変数（`--color-*`）を使ってテーマ対応を維持する

## 根拠

- ビュー間のナビゲーションは基本的な UI 要件である
- タブ UI はビュー切替のメンタルモデルと一致する
- 全ページ共通とすることで、どのビューからでも他のビューへ遷移可能になる
