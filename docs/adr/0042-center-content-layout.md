# 0042. コンテンツのセンタリング

## ステータス

承認済み (2026-04-16)

## コンテキスト

画面幅が広い環境（ワイドモニタ等）ではコンテンツが左寄せのまま横に広がり、可読性が低下する。
GitHub のようにコンテンツを中央寄せにし、最大幅を制限することで視認性を改善したい。

## 決定

- `main` 要素に `max-width: 1280px` と `margin: 0 auto` を設定する
- 1280px は GitHub のコンテナ全体幅と同等
- サイドバーのないレイアウトだが、コードやdiff表示で横幅を活用するためこの値を採用した
- `padding: 0 1rem` は従来通り維持し、狭い画面での余白を確保する

## 影響

- 1280px 以下の画面では表示に変化なし
- ヘッダーの sticky 指定は `main` 内に収まるため影響なし
